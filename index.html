RAM: The Computer's Short-Term Memory

Your computer's work desk

Think about: Why does your computer get slow when you open many Chrome tabs?

1950s

Magnetic cores

Woven by hand like necklaces

1980s

256KB chips

Cost like an iPhone today!

Today

16GB+ sticks

Stores 100,000 photos at once
What happens to RAM when you turn off the computer?
Everything gets erased
It saves to the disk
Stays the same

Fun Fact

The Apollo 11 computer (1969) had 64KB of RAM. Today a smartwatch has 100 times more!

Key Concept

RAM is volatile: Needs constant power. Like your thoughts when you're awake.

More RAM = More programs open at once without slowing down.

Conclusion: What We Learned

The Memory Journey Summary

From Magnetic Cores to Quantum Bits

Memory keeps getting faster, smaller, and smarter. Each type has its perfect use!

RAM is Fast

Short-term memory for active programs. More RAM = more multitasking.

HDD is Cheap

Best for storing lots of data you don't access daily. Great value.

SSD is Speedy

Perfect for your operating system and daily programs. Lightning fast!
